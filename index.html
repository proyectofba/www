Contenido del Curso:


Módulo 1.
Aprende a escoger los productos más rentables del Mercado para que tengas certeza que venderá.

El primer módulo contiene los mejores trucos y técnicas para escoger los productos con un alto margen de ganancia y más vendidos.

Contiene:

  • Cuales son los productos más rentables para vender.
  • Que productos debes evitar a todo costa.
  • Como usar la hoja de calculo (incluida) eficazmente.
  • Las mejores paginas para obtener ideas de productos.
  • Como saber que productos se venden hoy como pan caliente.
  • Los sectores específicos para importar productos regulados.
  • y mucho más.

Módulo 2.
Aprenderás encontrar productos de Calidad con Proveedores directos. Adiós intermediarios.

Te daremos las herramientas y guiones para no solo encontrar proveedores buenos y directos sino también como negociar excelentes términos con ellos.

Cuando hayas terminado el módulo sabrás:

  • Cómo filtrar los mejores proveedores para tu negocio.
  • Tips de negociación que usan los altos ejecutivos para obtener mejores términos de compra.
  • Cómo pagar a tu proveedor de forma segura para que nunca salgas estafado.
  • El arma secreta que uso para encontrar los mejores productos con mayor calidad que todavía no han llegado al mercado latino y con enorme potencial.
  • Como saber de antemano cuánto pagar de impuestos y requisitos para importar tu productos formalmente al país.

Módulo 3.
Cómo Importar tus productos al país 100% formal y con tus propios términos.

Se acabaron los secretos. En este módulo te enseño las tácticas que usamos para importar virtualmente cualquier productos al país legalmente y sin usar intermediarios.

Al finalizar Este módulo sabrás:

  • Cómo determinar los impuestos que tienes que pagar al momento de hacer tus importaciones.
  • Cómo buscar buenos agentes aduanales para tu negocio para que formes una relación provechosa a largo plazo.
  • Como solicitar sus servicios eficazmente para que nunca te rechacen y quieran trabajar contigo.
  • Cómo determinar la fracción arancelaria de tus productos para que nunca te sorprenda la aduana.
  • Cómo darse de alta eficazmente paso a paso ante hacienda con ejemplo práctico como si estuvieras viéndome hacerlo a lado de ti.
  • Qué son y cómo funcionan los padrones sectoriales para importar artículos especiales al país.

Módulo 4.
Cómo Preparar tus productos para Amazon FBA.

Este es un tema del que se habla mucho pero se hace poco. Con nuestros ejemplos prácticos sabrás donde comprar tus códigos de barras aun precio casi regalado para que no tengas que meterte en trámites.

Esto es lo que aprenderás:

  • Códigos de barras para tus productos (UPC, Códigos de Inventario) para que tu producto tenga su propio código de identificación universal.
  • Cómo crear los códigos de inventario en Amazon y las etiquetas de envío para tus productos.
  • Cómo calcular los costos de preparación y las tarifas de Amazon para que sepas de antemano cuánto vas a pagar y antes de hacer la importación y más...

Módulo 5.
Marketing y Ventas.

El marketing y las ventas son el alma de cualquier negocio y en este no es la excepción.

En este módulo aprenderás:

  • Crear descripciones que atraen a tus compradores y optimizar tus imágenes para maximas conversiones.
  • Las 4 estrategias que debes dominar. Esto te va a servir para maximizar conversiones (ventas) sin invertir demasiado dinero en marketing.
  • Como optimizar tus imágenes para que sobresalga de la competencia.
  • Como hacer los Títulos y descripciones basadas en beneficios de tu producto para que maximices las probabilidades de compra.
  • Ejemplos de Buenas vs Malas Publicaciones para que sepas identificar, corregir y optimizar tu publicación para máxima conversiones.
  • La lista de recursos y sitios donde puedes contratar estos servicios súper económicos para ahorrarte tiempo y dinero.
  • Cómo obtener reviews para crear prueba social inmediata.
